### **5. Konfiguration der Screenshot-Verzeichnisse**
```python
# /home/git/ProjektFitness/config.py
import os
from pathlib import Path

# Standardpfad mit Fallback
SCREENSHOT_DIR = os.getenv(
    "SCREENSHOT_DIR",
    "/home/bastian/ownCloud/Persönlich/coffeeproject/Projekt Fitness/Strava_Screenshots"
)

# Validierung des Pfades
def validate_screenshot_dir():
    path = Path(SCREENSHOT_DIR)
    if not path.exists():
        raise FileNotFoundError(f"Screenshot-Verzeichnis nicht gefunden: {path}")
    if not path.is_dir():
        raise NotADirectoryError(f"Pfad ist kein Verzeichnis: {path}")
    return path
```

## **Risikominimierung**

| **Risiko**                          | **Konkrete Lösung**                                                                                     |
|--------------------------------------|--------------------------------------------------------------------------------------------------------|
| Datenverlust bei `rsync`             | Trockenlauf (`--dry-run`) + Backup des Quellverzeichnisses.                                            |
| Berechtigungsprobleme                | Explizites `chown` und `chmod` für `/home/git/ProjektFitness`.                                         |
| Pfadkonflikte in Dokumentation      | Backup der Dokumentation + Validierung mit `grep` nach dem Ersetzen.                                   |
| Datenbank nicht erreichbar          | Pfad in Konfiguration relativ (`./database/fitness.db`) + Test der Verbindung.                        |
| Fehlende Umgebungsvariablen         | Fallback-Werte in `config.py` + Validierungsfunktion für Pfade.                                       |
